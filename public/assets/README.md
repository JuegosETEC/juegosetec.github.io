# Recursos Compartidos

Colocá aquí los recursos compartidos entre múltiples juegos, como:

- Fuentes (fonts)
- Imágenes de fondo comunes
- Archivos de audio compartidos
- Hojas de estilos CSS compartidas
- Cualquier otro recurso referenciado por múltiples juegos

## Ejemplo:
\`\`\`
/public/assets/
  ├── fonts/
  │   └── game-font.ttf
  ├── sounds/
  │   └── background-music.mp3
  └── images/
      └── common-background.png
